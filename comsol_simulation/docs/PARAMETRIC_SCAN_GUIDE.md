## 🔧 操作步骤

### 第一组: 速度 0.1 cm/s, 宽度 150μm

**1. 打开现有模型**
```
文件 → 打开 → microfluidic_chip.mph
```

**2. 修改参数**
```
在模型树中点击 "全局定义" → "参数"

修改以下参数:
- 将 v_in 的值改为 0.0001 (对应0.1 cm/s)
- 将 W 的值改为 0.15 (对应150 μm)
```

**3. 更新几何**
```
在模型树中点击 "几何1"
右键 → "重建几何"
或
点击"几何"选项卡 → "重建"
```

**4. 更新网格**
```
在模型树中点击 "网格1"
右键 → "重建网格"
等待网格生成完成
```

**5. 运行求解**
```
点击"计算"按钮 (绿色三角)
等待求解完成 (检查日志和收敛信息)
```

**6. 导出数据**
```
右键点击 "导出" → "数据"
设置:
  - 数据集: 研究1/稳态1
  - 几何实体层次: 域
  - 要输出的变量: u, v, p
保存为: v0.1_w150.csv
```

**7. 保存模型**
```
文件 → 另存为 → v0.1_w150.mph
```

**8. 重复步骤1-7**，依次完成所有9组

## 📊 验证标准

每组数据导出后应验证:
```
✅ 速度 u: 应该有非零值
✅ 压力 p: 从入口到出口递减
✅ 无NaN或无穷值
✅ 文件大小合理 (10-20 MB)
```

## 🚨 常见问题

**Q: 求解失败或收敛慢**
A: 尝试减小速度参数或使用更粗的网格

**Q: 速度或压力为零**
A: 检查边界条件是否正确设置

**Q: 网格生成失败**
A: 重建几何后重新生成网格

## 💡 优化技巧

**批量操作**:
1. 可以同时打开多个COMSOL窗口
2. 修改参数 → 求解 → 导出 → 切换窗口
3. 提高效率

**模型保存**:
1. 每组完成后立即保存
2. 使用清晰的命名约定
3. 避免覆盖之前的模型
